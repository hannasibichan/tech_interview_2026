## How to Clone and Contribute

### 1. Fork the Repository

Go to the original repo and click the **Fork** button at the top right to copy it to your GitHub account.

Original repo: https://github.com/FOSSCLUB-LBSITW/tech_interview_2026

### 2. Clone to Your Local System

```bash
git clone https://github.com/<your-username>/tech_interview_2026.git
```

```bash
cd tech_interview_2026
```

### 3. Make Changes

Open the folder in VS Code or any editor and make your changes to `index.html`, `style.css`, etc.

### 4. Add, Commit, and Push

```bash
git add .
```

```bash
git commit -m "your commit message here"
```

```bash
git push origin main
```

### 5. Create a Pull Request

- Go to your forked repo on GitHub
- Click **Compare & pull request**
- Add a title and description
- Click **Create pull request**
